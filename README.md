# redmond-onboarding-flow
Associate onboarding flow and documentation

This page is to help us gather documentation regarding the onboarding flow and which teams have responsibility for which tasks. The flow consists of the time from when an associate assigns their offer letter through their 90-day mark. This is being hosted here so that I can share it with multiple people and they can review the buckets of tasks. We can make quick changes and push those out to the group.
